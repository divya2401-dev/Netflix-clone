# 🎬 Netflix Clone

A fully functional **Netflix-inspired front-end clone** built using HTML and CSS.
This project replicates the core UI and navigation experience of Netflix, including a profile selection screen, multiple content pages, and categorized sections.

## ✨ Features

* 👤 **Profile Selection Screen (Landing Page)**

  * Just like Netflix, the app opens with a user profile screen

* 🏠 **Home Page**

  * Continue Watching
  * Trending Now
  * Action Movies
  * Drama
  * Romance

* 📺 **Shows Page**

  * US TV Dramas
  * Award-Winning Shows
  * Top 10 in India Today
  * Teen & High School Shows

* 🎬 **Movies Section**

* 🎮 **Games Section**

* 📰 **News & Popular Section**

* ❤️ **My List (Saved Content)**

* 🌐 **Browse by Language**

* 🎥 **Hero Banner Section**

  * Large featured content banner on each page (Netflix-style)

* 🔗 **Multi-Page Navigation**

  * Smooth navigation between all sections

## 🎯 How It Works

1. The project opens with a **profile selection screen**
2. Clicking a profile redirects to the **Home page**
3. Users can navigate across different sections like Netflix
4. Each page contains categorized content rows and a hero banner
